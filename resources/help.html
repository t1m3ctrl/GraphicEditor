Справка по программе

Основные функции

  • Добавление функций: Вы можете ввести выражение функции и добавить его в список для дальнейшего отображения.
  • Редактирование графиков: Для каждой функции можно настроить цвет, стиль линии и толщину.
  • Сохранение и загрузка данных: Сохранение текущих настроек и графиков в файл, а также загрузка ранее сохранённых данных.
  • Изменение диапазонов осей: Вы можете настроить диапазоны осей для точности отображения данных.

Математические функции

Программа позволяет вводить математические выражения для отображения графиков. Поддерживаемые функции включают базовые арифметические операции, а также более сложные математические выражения.

Используемые математические функции:

Программа поддерживает следующие функции, которые можно использовать при вводе математических выражений:

  • abs(x) — возвращает абсолютное значение числа x.
  • acos(x) — арккосинус числа x, результат в радианах.
  • asin(x) — арксинус числа x, результат в радианах.
  • atan(x) — арктангенс числа x, результат в радианах.
  • atan2(x, y) — арктангенс с двумя аргументами, который возвращает угол между осью X и точкой (x, y).
  • ceil(x) — округляет число x в большую сторону.
  • cos(x) — косинус числа x (в радианах).
  • cosh(x) — гиперболический косинус числа x.
  • e — математическая константа e, основание натурального логарифма (примерно 2.718).
  • exp(x) — экспоненциальная функция, возвращает e^x.
  • fac(x) — факториал числа x (для целых чисел).
  • floor(x) — округляет число x в меньшую сторону.
  • ln(x) — натуральный логарифм числа x (основание e).
  • log(x) — десятичный логарифм числа x (если не используется TE_NAT_LOG), иначе — натуральный логарифм.
  • log10(x) — десятичный логарифм числа x (по основанию 10).
  • ncr(x, y) — биномиальный коэффициент "из x по y", то есть количество сочетаний.
  • npr(x, y) — количество размещений, то есть x элементов, взятых по y.
  • pi — математическая константа π, отношение длины окружности к её диаметру (примерно 3.14159).
  • pow(x, y) — возводит число x в степень y.
  • sin(x) — синус числа x (в радианах).
  • sinh(x) — гиперболический синус числа x.
  • sqrt(x) — квадратный корень из числа x.
  • tan(x) — тангенс числа x (в радианах).
  • tanh(x) — гиперболический тангенс числа x.

Примеры ввода:

  • x^2 + 3*x - 5 — квадратичная функция, отображающая параболу.
  • sin(x) — синусоида, периодическая функция.
  • log(x) — натуральный логарифм.
  • x^3 - 6*x^2 + 11*x - 6 — кубическая функция.
  • cos(x) * sin(x) — произведение синуса и косинуса, отображающее осцилляции.

Ограничения программы:

  • Невозможно использовать некоторые сложные математические функции, которые не поддерживаются библиотекой tinyexpr.
  • Будьте осторожны с делением на ноль и другими математическими ошибками. Программа заменит такие значения на 0 или другое дефолтное значение.

Редактирование графиков

Каждый график можно редактировать по нескольким параметрам:

  • Цвет линии: Вы можете изменить цвет линии графика через контекстное меню. Выберите пункт "Изменить цвет линии" и выберите цвет из палитры.
  • Стиль линии: Стиль линии можно изменить на одну из нескольких предустановленных опций, таких как "Сплошная", "Штриховая", "Точечная" или "Штрих-точка".
  • Толщина линии: Для каждой функции можно установить толщину линии от 1 до 5.

Работа с файлами

Для сохранения и загрузки данных можно использовать меню "Файл". Все функции и их настройки сохраняются в файл с расширением .funcplot.

  • Сохранить: Сохранение текущего проекта в файл.
  • Открыть: Загрузка ранее сохранённого проекта из файла.
  • Новый файл: Очистка текущего проекта и создание нового.
  • Сохранить как: Сохранение проекта под новым именем.
