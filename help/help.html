Справка по программе

Эта программа позволяет визуализировать графики математических функций.

Основные функции

  • Добавление функций с использованием текстового поля ввода.
  • Изменение параметров графиков (цвет, стиль, толщина).
  • Сохранение и загрузка графиков.

Навигация

Для перемещения по справке используйте ссылки:

  • Как добавить функцию
  • Как редактировать график
  • Как сохранить работу

Как добавить функцию

Введите выражение функции в текстовое поле и нажмите кнопку "Добавить".

Как редактировать график

Щелкните правой кнопкой мыши на графике, чтобы открыть параметры редактирования.

Как сохранить работу

Используйте меню "Файл" → "Сохранить", чтобы сохранить текущий проект.
